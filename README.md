# Brendon Earl - Personal Website

## Quick Start

```bash
#view versions and select latest
nvm ls
nvm use node
# Install dependencies
npm install

# Start dev server (http://localhost:4321)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
